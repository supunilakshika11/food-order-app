# 🍔 Online Food Ordering System (React + TypeScript)

## 📌 Project Description
This is a frontend food ordering system built using React, TypeScript, and Axios. It allows users to browse food items, add to cart, and place orders with authentication protection.

---

## 🚀 Features
- User Authentication (Login/Register)
- Food Listing from API
- Add to Cart system
- Cart Quantity management
- Checkout / Order system
- Protected Routes (JWT style)
- Responsive UI

---

## 🛠️ Tech Stack
- React (TypeScript)
- React Router
- Axios
- Context API
- CSS (Inline / Tailwind optional)
